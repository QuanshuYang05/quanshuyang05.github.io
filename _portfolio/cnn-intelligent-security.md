## 基于 CNN 的智能防控系统

**角色**：系统集成工程师 | **时间**：2025.01 - 2025.04

大学生创新创业训练计划项目，基于深度学习的智能安全防控系统。

### 核心功能

- **人体异常动作识别**：基于 CNN 的暴力行为检测
- **陌生人脸检测**：深度学习人脸识别与陌生人告警
- **火灾探测**：实时视频流火灾检测
- **模块协同**：多模块数据交互与统一告警输出

### 主要工作

1. 负责将异常动作识别、人脸检测、火灾探测等模块集成至同一系统
2. 确保模块间数据交互顺畅，实现系统协同运作
3. 编写系统开发文档，明确架构与接口

### 技术栈

CNN · OpenCV · Python · 多模态融合
